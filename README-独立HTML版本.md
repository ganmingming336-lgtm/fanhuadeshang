# 彩票分析器 - 独立HTML版本

## 📋 简介

这是一个完全独立的HTML文件，包含了彩票号码分析的所有功能。您只需要下载一个HTML文件，就可以在任何现代浏览器中离线使用，无需任何技术知识。

## 🚀 使用方法

### 1. 下载和打开
- 下载 `lottery-analyzer.html` 文件
- 双击文件在浏览器中打开（或右键选择"打开方式"）
- 支持的浏览器：Chrome、Firefox、Safari、Edge等

### 2. 输入数据
- 在左侧输入框中粘贴您的彩票号码数据
- 支持的格式：
  - 逗号分隔：`5, 12, 23, 34, 45, 49`
  - 空格分隔：`1 2 3 4 5 6`
  - 换行分隔：每行一组号码
  - 混合格式：任意组合

### 3. 分析号码
- 点击"分析号码"按钮
- 实时查看分析结果

## 🎯 功能特性

### ✅ 完整功能
- **数据验证**：实时检测无效字符
- **频率计算**：分析1-49范围内每个号码的出现频率
- **热号/冷号分析**：自动识别最频繁和最不频繁的号码
- **统计摘要**：显示总抽奖次数、总号码数等关键指标
- **频率图表**：可视化展示所有号码的频率分布
- **详细表格**：按排名显示每个号码的完整统计信息

### 📊 导出功能
