# 12生肖预测工具 - Zodiac Predictor

## 简介

这是一个完全独立的HTML工具，用于分析12生肖的历史数据，使用多种预测算法推算下一期最可能出现的生肖。

## 文件位置

- `zodiac-predictor.html` - 单个HTML文件，包含所有功能

## 功能特性

### 1. 生肖数据管理
- **支持12个生肖**: 鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪
- **多种输入格式**: 支持编号（1-12）或生肖名称
- **灵活分隔符**: 支持逗号、空格、换行、中文逗号、顿号等分隔
- **数据验证**: 自动检查输入是否为有效生肖，提供错误提示
- **本地存储**: 使用localStorage自动保存历史数据，无需登录

### 2. 五种预测算法
1. **频率分析**: 统计每个生肖的出现频率，频率越高预测概率越高
2. **冷热号法**: 分析最近N期数据，长期未出现的生肖（冷号）可能反弹
3. **周期性分析**: 根据平均出现间隔判断每个生肖的"到点"概率
4. **组合概率**: 基于Markov转移矩阵和共现关系，考虑生肖之间的关联
5. **移动平均**: 使用指数加权移动平均，近期数据权重更高

### 3. Top 3 推荐卡片
- **醒目展示**: 最可能出现的前3个生肖，采用渐变色背景
- **详细指标**: 
  - 综合预测概率（百分比）
  - 历史出现次数和频率
  - 最后出现间隔期数
  - 平均出现间隔期数
- **预测原因**: 基于多算法的智能分析，解释为什么推荐这个生肖

### 4. 12生肖完整统计表
- **按出现次数排序**: 一目了然地看到热号和冷号
- **完整数据**: 生肖名、编号、出现次数、频率、最后间隔、平均间隔
- **冷热状态标签**: 热号（红）、冷号（蓝）、常规（灰）
- **统计概览**: 显示最常出现和最少出现的生肖

### 5. 频率柱状图
- **可视化展示**: 12个生肖的出现次数柱状图
- **颜色编码**:
  - 绿色: Top 3 推荐生肖
  - 橙色: 热号（高于平均1.2倍）
  - 灰色: 冷号（低于平均0.6倍）
  - 蓝色: 常规生肖
- **交互效果**: 鼠标悬停显示详细数据，带动画效果

### 6. 分析结果展示
- **五种算法各自的Top 3**: 分别展示每种算法的预测结果
- **综合预测Top 3**: 根据权重加权平均后的最终推荐
- **完整排名表**: 可选择显示Top K（默认12个）
- **可视化进度条**: 直观显示每个生肖的预测概率

### 7. 高级功能
- **权重自定义**: 调整5种算法的权重，自动归一化
- **参数调节**: 
  - 冷热号分析的最近N期
  - 移动平均的衰减系数α
  - 输出结果的Top K数量
- **导出功能**:
  - JSON格式: 包含完整配置和12生肖详细数据
  - CSV格式: 方便在Excel中分析
  - 历史数据备份: JSON格式导出
- **临时分析**: "分析时包含输入框内容"选项，可测试数据而不保存

## 使用方法

### 方式一：直接打开
1. 双击 `zodiac-predictor.html` 文件
2. 在浏览器中打开（推荐Chrome、Firefox、Edge）

### 方式二：输入示例数据
1. 点击"填充示例数据"按钮
2. 点击"立即分析预测"查看结果

### 方式三：输入自己的数据
1. 在"历史开奖数据"输入框中粘贴数据，例如：
   ```
   鼠 牛 虎
   兔 龙 蛇
   1 2 3
   4,5,6
   ```
2. 点击"添加到历史数据"保存到本地
3. 点击"立即分析预测"查看预测结果

### 数据格式示例

支持多种格式，自动识别：

```
# 生肖名称 + 空格
鼠 牛 虎

# 生肖名称 + 逗号
兔,龙,蛇

# 编号 + 空格
1 2 3

# 编号 + 逗号
4,5,6

# 混合格式
鼠 2 虎,4 龙
```

生肖编号对应：
- 1-鼠, 2-牛, 3-虎, 4-兔, 5-龙, 6-蛇
- 7-马, 8-羊, 9-猴, 10-鸡, 11-狗, 12-猪

## 结果解读

### Top 3 推荐卡片
- **第1名（绿色）**: 综合概率最高，最推荐
- **第2名（蓝色）**: 综合概率次高
- **第3名（橙色）**: 综合概率第三

### 预测等级
- **热号**: 最近出现频率很高
- **冷号**: 长期未出现，可能反弹
- **常规**: 出现频率正常

### 预测原因
工具会自动分析并说明为什么推荐某个生肖，例如：
- "综合概率达X%，高于平均"
- "历史出现频率高"
- "已N期未出，可能反弹"
- "周期性分析显示'到点'概率高"
- "与最近一期组合关系强"

## 算法参数说明

### 冷热号：最近 N 期
- 默认值: 20
- 范围: 1-9999
- 说明: 分析最近N期数据来判断冷热号，N越小越关注短期趋势

### 移动平均：衰减系数 α
- 默认值: 0.95
- 范围: 0.7-0.999
- 说明: 越接近1越看重长期数据，越小越偏重近期数据

### 输出：Top K
- 默认值: 12（显示全部）
- 范围: 3-12
- 说明: 在完整排名表中显示前K个生肖

### 综合权重
- 默认: 5种算法等权（各1.0）
- 可调整: 根据经验调整各算法权重
- 自动归一化: 总和自动调整为1.0

## 常见问题

### Q: 数据会丢失吗？
A: 数据保存在浏览器的localStorage中，除非清除浏览器数据或手动删除，否则不会丢失。建议定期导出备份。

### Q: 如何清空所有数据？
A: 点击"清空历史数据"按钮，会清除所有保存的历史数据。

### Q: 为什么预测结果概率都很接近？
A: 这是正常现象。如果历史数据中各生肖出现比较均匀，预测概率就会接近。数据越不均匀，预测差异越大。

### Q: 哪种算法最准确？
A: 没有绝对准确的算法。建议使用"综合预测"（默认等权），它结合了5种算法的优势。也可以根据实际情况调整权重。

### Q: 可以在手机上使用吗？
A: 可以。界面采用响应式设计，在手机浏览器中打开即可使用。
